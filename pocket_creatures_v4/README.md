# Pocket Grove v4

ブラウザで遊べる、オリジナルのモンスター収集RPGです。

## 今回の更新点

- BGM を追加
  - Web Audio API で鳴る、オリジナルのチップチューン風 BGM を実装
  - タイトル / フィールド / バトル / クリアで曲調が切り替わります
  - ブラウザの仕様上、最初のキー入力やクリックの後に再生されます
  - 右側の「BGM ON / OFF」ボタンで切り替えできます
  - ON / OFF の設定はブラウザに保存されます
- パッケージを最終化
  - 単体 HTML 版を整理して ZIP に同梱
  - README も更新
- 細かな安定化
  - 戦闘中にセーブされたデータを読み込んだ場合でも、探索状態で安全に再開するよう補正
  - 既存の視認性改善、中央表示、PC 向けレイアウト調整はそのまま維持

## 遊び方

### 単体 HTML 版

`pocket_grove_singlefile_v4.html` をそのままブラウザで開いて遊べます。

### 操作

- 移動: 矢印キー / WASD / 画面上の十字キー
- 決定・会話: Enter / Space / A ボタン
- パーティ: P / B ボタン / START
- 手動セーブ: SELECT / サイドバーのセーブボタン
- BGM 切替: 右サイドバーの `BGM ON / OFF`

## メモ

- これは権利面で安全なオリジナル作品です
- Nintendo / Game Freak / Creatures とは無関係です
- 一部ブラウザ設定や省電力設定では、音が鳴るまでに最初の操作が必要です
