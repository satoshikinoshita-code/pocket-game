# Pocket Grove v7

Pocket Grove の改善版です。今回は、依頼どおり **実装チェック → 進化システム追加 → 再デバッグ / リファクタリング** の順で進めています。

## 今回の実装計画

1. 既存 v6 を対象に、構文・主要導線・セーブ互換の観点で不具合チェックを行う
2. 問題が大きくなければ、モンスターの進化システムを追加する
3. 旧セーブ互換、図鑑、捕獲、終盤導線との整合をとる
4. 最後にデバッグと軽いリファクタリングを行い、ZIP 版として最終化する

## 実施したチェック

- HTML / JS の読み込み確認
- 新規開始 → 初回戦闘 → 勝利までの動作確認
- 捕獲 → 仲間化 → 図鑑反映の確認
- セーブ / ロードの往復確認
- 旧版セーブ相当データの読込確認
- エンディング / Hall of Fame の表示確認

大きな実装破綻は見つからなかったため、そのまま進化システムを追加しました。

## v7 の追加 / 改善内容

### 1. 進化システム

- 一定レベルでモンスターが進化
- 進化演出用のオーバーレイを追加
- 進化後は見た目用カラー、能力値、覚える技を更新
- 進化した姿も図鑑に記録

### 2. 進化対応モンスター

- フレアコ → フレアロウ
- モスリン → モスリオン
- ブルキット → ブルクレスト
- ザプップ → ザプライド
- ペブラ → ペブロック
- ペタリア → ペタローズ
- ボルタフ → ボルトラス
- クラグーン → クラガイア
- シンダル → シンダリオ
- スラゴン → スラヴァーン
- ケルパプ → ケルフィン
- タイデル → タイドレア
- グリムキット → グリムレイ
- ノクテム → ノクステラ

### 3. バグ修正 / 安定化

- 旧セーブ読込時、進化条件を満たしている仲間を自動で整合
- セーブ読込時に、能力値 / 技構成を最新版定義へ正規化
- 高レベルで生成される野生 / 敵モンスターが、進化済みの姿で自然に出現するよう調整
- 進化演出中は入力が暴発しないよう制御を追加
- セーブバージョンを v7 に更新しつつ、v6 以前の読込にも配慮

### 4. 軽いリファクタリング

- 進化判定・進化適用・能力値再計算をヘルパー化
- セーブ互換処理を v7 用に整理
- 図鑑表示に進化情報を追加
- パーティ表示に「進化可 / 進化済み」チップを追加

## 操作

- 移動: 矢印キー / WASD
- 調べる: Enter / Space / A ボタン
- パーティ: P / START
- 図鑑: K
- セーブ: SELECT / 手動セーブボタン

## 遊び方

1. `pocket_grove_singlefile_v7.html` をブラウザで開く
2. はじめから開始してスターターを選ぶ
3. 野生モンスターを弱らせ、キャプチャボールで仲間にする
4. レベルを上げると、一部モンスターは進化する
5. 四天王 4 人とラスボスを倒すと、エンディングと殿堂入りへ進む

## セーブについて

- セーブはブラウザの `localStorage` を使用します
- サーバーや外部データベースは不要です
- 同じブラウザ・同じ保存領域なら続きから遊べます
- v6 以前のセーブも読み込めるようにしてあります
- 旧セーブ読込時、進化条件を満たしている仲間は自動で新仕様へ整合されます

## 補足

- BGM は Web Audio API で鳴っています
- ブラウザの仕様上、最初のクリックやキー入力のあとに再生されます
- Nintendo / Game Freak / Creatures とは無関係のオリジナル作品です
